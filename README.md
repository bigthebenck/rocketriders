# Rocket Riders

A Minecraft datapack collection that adds rocket-powered combat and racing game modes.

Compatible with pack formats 88–101 (Minecraft 1.21.x), with overlays for 1.21.10 and 1.21.11.

## Datapacks

| Pack | Description |
| --- | --- |
| `rocketriders` | Core datapack — required by all game modes. |
| `rocketriders_experimental` | Additional experimental features. |
| `rr_normal` | Normal game mode. |
| `rr_chase` | Chase game mode. |
| `rr_crusade` | Crusade game mode. |
| `rr_ctf` | Capture the Flag game mode. |
| `rr_duel` | 1v1 Duel game mode. |
| `rr_powerups` | Powerups game mode. |
| `rr_swap` | Swap game mode. |
| `rr_sandbox` | Sandbox mode for free play and testing. |

## Installation

1. Copy the desired pack folders into your world's `datapacks/` directory.
2. Always include `rocketriders` (the core pack) alongside any game-mode pack.
3. In-game, run `/reload` or rejoin the world.
